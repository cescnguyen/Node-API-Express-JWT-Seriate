Authentication

Authentication is a bearer token system. Tokens are obtained by authenticating with a valid username and password to /api/authenticate.

When a token is obtained, it is valid for 30 minutes. The token must be sent as a header key/value pair in the request: Key: x-access-token, Value: <token>
